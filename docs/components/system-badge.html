SystemBadge

Fixed NEW/BETA label badge — matches Figma's cm_global-badge exactly. Fixed label text and fixed colors, not a generic customizable badge. Renamed from Badge when the general-purpose Badge component (Figma node 76:4205, also literally named "Badge") took over the canonical name.
