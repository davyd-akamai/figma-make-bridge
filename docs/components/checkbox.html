Checkbox

Checkbox with a real indeterminate state, small/large sizes. Native <input type="checkbox"> visually hidden, custom-drawn box as its sibling — real keyboard/click/form semantics for free.

Figma node 58:4716 import { Checkbox } from "figma-make-bridge" Source ↗

Playground

Click the checkbox itself, or toggle the states below. indeterminate is a plain prop — like the native DOM property, clicking does not clear it automatically.

Select-all pattern

The parent's indeterminate is recomputed from its children's checked state on every change — matching the note in GUIDELINES.md that a "select all" parent must own this, not the checkbox itself.
