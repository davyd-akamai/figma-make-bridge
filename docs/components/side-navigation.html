SideNavigation

Collapsible left nav rail — unpinned rests collapsed (48px, icon-only) and expands to 232px on hover; pinned locks the rail to a fixed state and ignores hover. Accordion sections, page selection, exact-vector icons, NEW/BETA badges.

Figma node 17:2719 import { SideNavigation } from "figma-make-bridge" Source ↗

Unpinned (hover to expand)

Rests collapsed at 48px. Hover anywhere over the rail to expand it to 232px; move away to collapse again. Click a section header to open/close it, click a page row to select it (its parent section header turns blue).

Pinned + expanded

Pinned locks the rail — hover no longer has any effect. The pin icon fills solid to indicate the locked state.

Pinned + collapsed (mini rail)

Pin doesn't only mean "locked open" — pinnedExpanded=false locks a permanently-collapsed, icon-only rail. Click the pin to unpin and confirm hover takes over again.
